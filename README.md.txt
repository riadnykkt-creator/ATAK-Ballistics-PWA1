# ATAK Ballistics - PWA → APK

## Jak vytvořit APK

1. Nahraj tento projekt na GitHub.
2. Vytvoř nový repozitář a nahraj všechny soubory.
3. Otevři záložku Actions → aktivuj workflow `Build APK`.
4. Počkej, až workflow dokončí.
5. Stáhni APK přímo z artifactů workflow.
6. Nainstaluj do telefonu.

✅ Hotová APK aplikace bude obsahovat mapu, sniper UI, balistiku, vítr a drop/drift výpočty.
